Investigation Overview¶

Ford GoBike System dataset includes information about individual rides made in a bike-sharing system covering the greater San Francisco Bay area. Please, find the dataset here

Data of each trip includes:

  • Trip Duration (seconds)
  • Start Time and Date
  • End Time and Date
  • Start Station ID
  • Start Station Name
  • Start Station Latitude
  • Start Station Longitude
  • End Station ID
  • End Station Name
  • End Station Latitude
  • End Station Longitude
  • Bike ID
  • User Type (Subscriber or Customer Subscriber = Member or Customer = Casual)

The goal of our analysis is to respond ton the following questions:

  • 1- How does the number of trips vary according to week day?
  • 2- How does the number of trips vary according to hour ?
  • 3- How does the duration vary according to user type ?
  • 4- How does the duration vary according to user's day of birth?
  • 5- What is the percentage of male, female and oder?
